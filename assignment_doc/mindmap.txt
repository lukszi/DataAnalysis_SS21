Einleitung
	Was ist das Projekt?
		Auswertung von reddit Posts des WallStreetBets reddits
		In denen werden Stocks genannt für die Werbung gemacht wird
		Diese kann man liken kommentare hinzufügen usw.
		Das Projekt filtert die Stock erwähnungen raus und guckt dann
		ob dieser Stock ab Zeitpunkt des Reddit Posts ansteigt
		Wir suchen weiterhin Korrelationen zwischen anzahl der kommentaren
		anzahl der Likes mit dem anstieg/fall der stock daten
	Motivation
		Vor einem halben jahr wurde eine Aktie namens Gamestop durch reddit posts manipuliert
		Many users commented about that stock and many started buying it so it went up
		Now we wanna check if these Reddit posts have an efect on other stocks.
Vorgehen
	Ideen
		Using the reddit Api and regex filter to extract stocks from a reddit post
		Using yahoo finance api to get the stock data to the stock mentioned in the reddit post
		With all these apis we collected all nesessary data to analyse if the reddit post have an effect on the stock
